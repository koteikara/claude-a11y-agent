# 用語集

Claude-A11y Agent でよく使う言葉を、できるだけ専門用語を避けて説明します。

| 用語 | やさしい説明 |
|---|---|
| アクセシビリティ | 年齢、障害、使う機器に関係なく、できるだけ多くの人が情報を受け取れるようにする考え方です。このツールでは、読み上げソフトでもページ内容が伝わるようにすることを重視します。 |
| 読み上げソフト（スクリーンリーダー） | 画面の文字やボタン、リンクを音声で読み上げるソフトです。目の見えない人や画面を見づらい人が使います。 |
| 代替テキスト（`alt`） | 画像の意味を文字で説明する情報です。画像が見えない人にも、何の画像か伝えるために使います。 |
| 表のタイトル（キャプション） | 表が何を示しているかを説明する短い題名です。読み上げソフトでも表の目的をつかみやすくなります。 |
| 「こちら」リンク問題 | リンク文が「こちら」「ここ」「詳細」だけだと、リンクだけを読んだときに行き先が分からない問題です。「申請書をダウンロードする」のように、リンク文だけで意味が分かる表現が望ましいです。 |
| ジョブ | 1ページ分の処理依頼です。スプレッドシートの `Jobs` タブでは、基本的に1行が1ジョブです。 |
| キュー（`queued`） | 処理待ちの状態です。順番待ちの列に並んでいるイメージです。 |
| `old` | 移行前の元ページです。料理にたとえると材料です。 |
| `ai` | AI がアクセシビリティ対応を入れた下書きです。料理にたとえると下ごしらえです。 |
| `gold` | 人が確認して承認した最終版です。料理にたとえると仕上げです。 |
| 要確認（`needs_review`） | AI が「人に見てほしい」と判断した状態です。失敗ではなく、確認が必要という意味です。 |
| 承認 | 内容を確認し、最終版としてよいと判断することです。Web管理画面では `承認（gold確定）` を押すと、gold が確定します。 |
| CMS | ホームページを管理する仕組みです。市の担当者がページを作成・更新するための管理システムを指します。 |
